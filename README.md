# Rishaan Yadav — Portfolio Website

A responsive personal portfolio website designed to showcase my skills, projects, achievements, and professional experience in Computer Science and Data Science. This website serves as an online resume and portfolio to highlight my technical expertise and work.

## Features

- Responsive design optimized for desktop, tablet, and mobile devices  
- Clean and modern professional layout  
- Interactive sections including Home, About, Skills, Projects, Achievements, Resume, and Contact  
- Project showcase with detailed pages for featured work  
- Contact form integrated with EmailJS for direct messaging  
- Social media integration including GitHub, LinkedIn, and ORCID  

## Technologies Used

- HTML5  
- CSS3  
- JavaScript (ES6+)  
- Bootstrap Icons  
- EmailJS (for contact form functionality)  
- Google Fonts  
- AOS (Animate On Scroll library)  
- Swiper JS (for project sliders)  

## Usage

This portfolio website is designed to present my professional profile and technical work. Visitors can:

- Learn about my background and expertise  
- View my technical skills and areas of specialization  
- Explore my featured projects  
- Download my resume  
- Contact me directly through the integrated contact form  

## Customization

The website is structured for easy customization. Styling and layout settings can be modified in the main CSS file, allowing adjustments to colors, fonts, and layout elements.

## 1. Project Overview

Build a modern, fully responsive personal portfolio website for Rishaan Yadav — a Computer Science Engineering graduate specializing in Machine Learning, AI, and Data Science. The site must look polished on both desktop and mobile, have smooth navigation, and allow visitors to contact Rishaan directly via an email form without leaving the site.

---

## 2. Tech Stack

- **HTML5, CSS3, Vanilla JavaScript** (no heavy frameworks needed)
- **EmailJS** for the contact form (sends email directly from the browser — no backend required)
- **Google Fonts** for typography
- **Font Awesome** or **Lucide Icons** for icons
- Fully **mobile responsive** using CSS Flexbox/Grid + media queries

---

## 3. Design Direction

- **Style:** Modern, clean, dark-accented with a professional feel. Think deep navy/charcoal background with bright accent color (e.g., electric blue `#0ea5e9` or teal `#14b8a6`).
- **Font pairing:** Display heading font (e.g., `Syne` or `DM Serif Display`) + clean body font (e.g., `DM Sans` or `Outfit`)
- **Animations:** Subtle fade-in on scroll for sections, smooth hover states on cards and buttons
- **NO generic purple gradient AI look**
- Mobile-first design — every section must look great at 375px width and above
